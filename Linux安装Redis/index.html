Linux安装Redis

Posted on 2019-06-23 | | Visitors

为第一次在linux下安装redis做个记录，并说明简单的配置。

安装步骤

  1. 下载安装包，以redis4.0.10为例。

  2. 将压缩包移动到指定目录并解压。

  3. 编译。

  4. 安装

  5. 配置文件redis.conf

redis.conf中都有一些注释说明各种配置的意义，还是比较好理解的。
几个常用的配置：

  • protected-mode 是否保护模式，保护模式对访问的ip和端口有限制
  • port 对外开放的端口
  • bind 限制访问的ip
  • timeout 连接超时后断开
  • daemonize 是否后台进程
  • requirepass 登陆密码
